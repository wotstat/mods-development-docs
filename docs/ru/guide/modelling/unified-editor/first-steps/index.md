# Знакомство c Unified Editor {#title}

Программа `Unified Editor` — редактор, официально распространяемый Lesta games для создания графических модификаций. С его помощью создаются визуальные эффекты, ангары и карты.

## Установка {#installation}
1. Скачайте последнюю версию `Unified Editor` из официальной [темы на форуме](http://forum.tanki.su/index.php?/topic/2199392-unified-editor-modders-edition/). Скачиваем последнюю, актуальную для игры установщик.

2. Скачанный установщик запускаем. Он имеет наименование по принципу `UnifiedEditor-ModdersEdition.X.XX.exe`, где `X.XX` - версия игры.

3. Выбираем язык для дальнейшей установки.

4. Принимаем Лицензионное соглашение.

5. Выбираем директорию, где будет установлен `Unified Editor`.

6. Нажимаем установить.

7. Ярлык после установки не создаётся. Переходим в директорию с установленным `Unified Editor`, в папку `.../unified_editor/`. Именно здесь и будет установлена программа `unifiededitor.exe`. По желанию, ярлык можно создать самому.

## Первый запуск {#first-run}
Запустив `Unified Editor` впервые, вас встретит `h08_mt_hangar` — стандартный ангар в игре, используемый вне событий по умолчанию.
